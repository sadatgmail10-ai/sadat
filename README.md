# Medic Dashboard Sample

یک پروژه نمونه React + Vite با بک‌اند Express برای داشبورد پزشکی.

## ویژگی‌ها

- رابط کاربری React 18
- رندر سمت سرور با Express
- ورود با حساب‌های دمو
- ذخیره توکن و نقش در `localStorage`
- PWA با `vite-plugin-pwa`
- قابل ساخت و استقرار روی هاست رایگان

## حساب‌های تست

- `sadat / sadat` (مدیر)
- `accountant1 / accountant123`
- `accountant2 / accountant123`
- `accountant3 / accountant123`
